To Run the Game:
	1. Clone the repo or download the files
	2. Run make.bat
	3. Run game.exe

How to Play:
	The goal of the game is to fire cannonballs and hit the targets.
	A point is scored for every hit target. Each time you miss the 
	target, you lose a life. When you run out of lives, the game is 
	over. Every 3 points earned, the cannon	power will be increased 
	by a small amount. There is no win condition. Players only try
	to achieve the highest possible score.

Controls:
	Rotate cannon.....................LEFT or RIGHT ARROW KEYS
	Shoot cannonball..................LEFT MOUSE BUTTON
	Restart game after loss...........R
	Pause.............................SPACE
